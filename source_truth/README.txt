Source Truth Folder

Put Jade Palace source truth .txt files here.

Preferred format:
KEY | SOURCE TEXT | SCREEN / LOCATION | TRANSLATOR NOTE

Example:
jppos.login.qr.preparing | Preparing POS login QR… | JPPOS / Login | Loading text before QR appears.

The translator website shows SOURCE TEXT to translators and keeps KEY for production JSON exports.

Update manifest.json when adding or removing files.

Translators do not upload source truth files. The website preloads the files listed in manifest.json.
